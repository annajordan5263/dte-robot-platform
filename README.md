# dte-robot-platform
# 深时探索者 (DeepTime Explorer, DTE) 仿生机器人算法平台

![DTE Logo](docs/images/dte-logo.png)

## 项目概述

深时探索者 (DeepTime Explorer, DTE) 是一个基于Onnia三叶虫仿生学的智能化机器人系统，旨在通过融合古生物学、仿生学、人工智能和脑机接口技术，构建能与人类思维深度协同、并理解复杂社会与文化背景的智能探索者。

## 核心特性

- **Onnia三叶虫仿生设计**：基于4.5亿年前晚奥陶纪三叶虫的形态与行为特征
- **多模态感知系统**：集成视觉、触觉、化学和地质传感器
- **AI自主决策**：基于强化学习和进化算法的智能决策系统
- **脑机接口(BCI)**：支持非侵入式意念控制和情感识别
- **人文社科知识库接口**：可接入语言学、历史学、社会学等知识库
- **分布式计算架构**：支持MPI和CUDA加速
- **多环境仿真**：支持Gazebo、MuJoCo和PyBullet仿真

## 技术栈

- **核心语言**：C++17
- **并行计算**：MPI, CUDA
- **API服务**：FastAPI, REST
- **数据库**：MySQL
- **仿真环境**：Gazebo, MuJoCo, PyBullet
- **部署平台**：ADLINK算法平台
